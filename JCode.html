1. A + B 问题

难度: 简单 标签: 基础, 数学 通过率: 85%

题目描述

给定两个整数 A 和 B，计算它们的和 A + B。

输入格式

输入包含两个整数 A 和 B，用空格分隔。

数据范围：-1000 ≤ A, B ≤ 1000

输出格式

输出一个整数，表示 A + B 的结果。

样例输入

输入 #1
3 4
输入 #2
-5 8

样例输出

输出 #1
7
输出 #2
3
代码编辑器
#include <iostream> using namespace std; int main() { int a, b; cin >> a >> b; cout << a + b << endl; return 0; }
